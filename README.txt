How to use this thing for your purposes:

Normally you would have to do this (but I included it):
{
    1. Download the Chrome executable from here: https://sites.google.com/a/chromium.org/chromedriver/downloads
    2. Add the driver to your current directory
}

Next:

3. Create and activate a virtualenv
4. Run pip3 install -r requirements.txt in the terminal 
5. Run file in terminal! (python3 wikiscaper.py)
